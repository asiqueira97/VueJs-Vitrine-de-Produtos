# Vitrine de Produtos com VUEjs

## Instalar dependências
```
npm install
```

### Executar projeto (dev)
```
npm run serve
```

### Visualizar projeto no navegador
```
Abra o navagador e acesse: http://localhost:8080/
```

### Compilar e minificar para ambiente de produção
```
npm run build
```
